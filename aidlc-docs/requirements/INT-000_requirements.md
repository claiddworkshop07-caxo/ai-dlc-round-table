# INT-000 Requirements: 備品貸出管理アプリ

## FR-001: 備品マスタ管理

### FR-001-1: 備品登録
- 管理者は備品名・カテゴリ・説明・保管場所を入力して備品を登録できる
- 登録時に一意のIDが自動採番される
- 入力: 備品名（必須）、カテゴリ（任意）、説明（任意）、保管場所（任意）
- 出力: 登録完了通知、備品ID

### FR-001-2: 備品編集・削除
- 登録済み備品の情報を編集できる
- 貸出中の備品は削除できない（エラーメッセージを表示）
- 返却済みの備品は論理削除（非表示）できる

### FR-001-3: QRコード発行
- 備品ごとにQRコードを生成・表示できる
- QRコードには備品IDが埋め込まれる
- QRコードはPNG形式でダウンロードできる
- QRコードは印刷用ページとして表示できる

### FR-001-4: 備品一覧
- 登録済み備品の一覧を表示する
- 貸出状況（貸出中/返却済み）を一目で確認できる
- 備品名・カテゴリでのフィルタリング・検索が可能

---

## FR-002: QRコードスキャンによる貸出・返却

### FR-002-1: QRコードスキャン
- スマートフォンのカメラでQRコードをスキャンできる（Webブラウザ対応）
- スキャン後、対象備品の情報と現在の貸出状況を表示する

### FR-002-2: 貸出処理
- 返却済みの備品をスキャンした場合、貸出フォームを表示する
- 入力: 借用者名（必須）、返却予定日（必須）、メモ（任意）
- 確認後、貸出記録を作成する
- 出力: 貸出完了通知、貸出ID

### FR-002-3: 返却処理
- 貸出中の備品をスキャンした場合、返却確認画面を表示する
- 確認後、返却記録を更新する（返却日時を自動記録）
- 出力: 返却完了通知

---

## FR-003: 貸出状況・ダッシュボード

### FR-003-1: 貸出状況一覧
- 現在貸出中の備品一覧を表示する（備品名・借用者・貸出日・返却予定日）
- 返却期限超過の備品をアラート表示（赤色ハイライト等）
- 返却期限が近い備品（例: 翌日まで）を警告表示（黄色ハイライト等）

### FR-003-2: 返却期限アラート
- 期限超過: 返却予定日を過ぎた貸出記録を強調表示
- 期限間近: 返却予定日まで1日以内の貸出記録を警告表示

---

## FR-004: 貸出履歴

### FR-004-1: 履歴一覧
- 全貸出記録（返却済み含む）を新しい順に一覧表示する
- 表示項目: 備品名、借用者名、貸出日時、返却予定日、返却日時、ステータス

### FR-004-2: 履歴検索・フィルタリング
- 備品名・借用者名での検索が可能
- 期間（貸出日）での絞り込みが可能
- ステータス（貸出中/返却済み）でのフィルタリングが可能

---

## データモデル（概要）

### 備品 (equipment)
| フィールド | 型 | 説明 |
|---|---|---|
| id | UUID | 一意ID |
| name | string | 備品名（必須） |
| category | string | カテゴリ |
| description | string | 説明 |
| location | string | 保管場所 |
| status | enum | available / borrowed |
| created_at | datetime | 登録日時 |
| deleted_at | datetime | 削除日時（論理削除） |

### 貸出記録 (lending_records)
| フィールド | 型 | 説明 |
|---|---|---|
| id | UUID | 一意ID |
| equipment_id | UUID | 備品ID（FK） |
| borrower_name | string | 借用者名（必須） |
| borrowed_at | datetime | 貸出日時 |
| due_date | date | 返却予定日（必須） |
| returned_at | datetime | 返却日時（返却後に設定） |
| memo | string | メモ |
| status | enum | active / returned |

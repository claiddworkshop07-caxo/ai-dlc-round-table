# INT-000 Intent: 備品貸出管理アプリ

## Purpose（目的）
社外ユーザーがスマートフォンのQRコードリーダーを使って備品の貸出・返却を記録できる管理アプリを構築する。
これにより、手作業による台帳管理を廃止し、備品の所在把握と業務効率化を実現する。

## Success Criteria（成功基準）
- 担当者がスマホでQRコードをスキャンするだけで貸出・返却を記録できる
- 備品マスタ（登録・編集・QRコード発行）をWebから管理できる
- 貸出中の備品一覧・返却期限アラートをリアルタイムで確認できる
- 貸出履歴をCSV等で参照・エクスポートできる

## Scope（スコープ）

### 対象
- 備品マスタ管理（備品の登録・編集・削除・QRコード発行）
- QRコードスキャンによる貸出・返却処理
- 貸出状況ダッシュボード（現在の貸出一覧・返却期限アラート）
- 貸出履歴一覧・検索

### 対象外
- ユーザー認証・権限管理（PoC段階ではスキップ）
- 備品の在庫数量管理（1点ずつの個品管理のみ）
- 外部システム連携（購買システム等）
- モバイルアプリ（ネイティブ）開発

## Target Users（対象ユーザー）
- 社外ユーザー（備品を借りる・返す利用者）
- 備品管理担当者（マスタ管理・状況確認）

## Tech Stack（技術スタック）
- Frontend: Next.js (TypeScript), Tailwind CSS, shadcn/ui
- Backend: Netlify Functions
- DB: Neon (PostgreSQL), Drizzle ORM
- Deploy: Netlify

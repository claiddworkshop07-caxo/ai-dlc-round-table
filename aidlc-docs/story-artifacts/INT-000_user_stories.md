# INT-000 User Stories: 備品貸出管理アプリ

## ユーザーストーリー一覧

| ID | タイトル | ユーザー | 優先度 |
|---|---|---|---|
| US-001 | 備品マスタの登録・編集・QRコード発行 | 備品管理担当者 | 最高 |
| US-002 | QRコードをスキャンして備品を借りる | 社外ユーザー | 高 |
| US-003 | QRコードをスキャンして備品を返却する | 社外ユーザー | 高 |
| US-004 | 貸出状況ダッシュボードで返却期限アラートを確認する | 備品管理担当者 | 高 |
| US-005 | 貸出履歴を検索・確認する | 備品管理担当者 | 中 |

---

## US-001: 備品マスタの登録・編集・QRコード発行
担当: UNIT-001 / Bolt-001

As a 備品管理担当者,
I want to 備品の情報を登録・編集し、QRコードを発行したい,
So that 備品にQRコードシールを貼り付けて、スキャン運用を開始できる。

詳細: `INT-000_US-001.md` 参照

---

## US-002: QRコードをスキャンして備品を借りる
担当: UNIT-001 / Bolt-002（予定）

As a 社外ユーザー（利用者）,
I want to スマートフォンでQRコードをスキャンして貸出手続きをしたい,
So that 受付担当者を介さずに自分で素早く借りることができる。

---

## US-003: QRコードをスキャンして備品を返却する
担当: UNIT-001 / Bolt-002（予定）

As a 社外ユーザー（利用者）,
I want to 返却時にQRコードをスキャンするだけで手続きを完了したい,
So that 返却手続きの手間を最小限にし、スムーズに業務に戻れる。

---

## US-004: 貸出状況ダッシュボードで返却期限アラートを確認する
担当: UNIT-001 / Bolt-003（予定）

As a 備品管理担当者,
I want to 貸出中の備品一覧と返却期限超過・期限間近のアラートを確認したい,
So that 未返却備品をタイムリーに把握し、督促対応ができる。

---

## US-005: 貸出履歴を検索・確認する
担当: UNIT-001 / Bolt-003（予定）

As a 備品管理担当者,
I want to 備品ごと・期間ごとの貸出履歴を検索・確認したい,
So that 備品の利用状況を把握し、管理業務の改善に活用できる。

# INT-000 US-001: 備品マスタの登録・編集・QRコード発行

## As a
備品管理担当者

## I want
- 備品の名称・カテゴリ・保管場所などの情報を登録・編集したい
- 登録した備品のQRコードを生成・印刷したい
- 備品一覧で貸出状況（貸出中/返却済み）を確認したい

## So that
備品にQRコードシールを貼り付け、スキャンによる貸出・返却運用を開始できる。
管理台帳の手作業をなくし、備品の所在を常に把握できる状態にする。

## Acceptance Criteria（受け入れ条件）

### 備品登録
- [ ] 備品名（必須）・カテゴリ・説明・保管場所を入力して備品を登録できる
- [ ] 登録後、一覧画面に新しい備品が表示される
- [ ] 登録時に一意のIDが自動採番される

### 備品編集
- [ ] 一覧から備品を選択し、各フィールドを編集・保存できる
- [ ] 貸出中の備品は削除できず、エラーメッセージが表示される
- [ ] 返却済みの備品は論理削除（非表示化）できる

### QRコード発行
- [ ] 備品詳細画面にQRコードが表示される
- [ ] QRコードにはその備品のIDが埋め込まれている
- [ ] QRコードをPNGファイルとしてダウンロードできる
- [ ] 印刷用QRコードページを開くことができる

### 備品一覧
- [ ] 登録済み備品の一覧が表示される（備品名・カテゴリ・ステータス）
- [ ] 「貸出中」「返却済み（利用可能）」のステータスが視覚的に区別できる
- [ ] 備品名・カテゴリでのキーワード検索・フィルタリングができる

## Notes
- QRコードライブラリ: `qrcode` (npm) を使用
- QRコードのフォーマット: `equipment:{id}` 形式のURLまたはJSON
